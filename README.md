<h2>🎓 Project Overview</h2>
<p>News platform designed to deliver articles and trending information in a fast, organized, and professional way. The website provides users with a comfortable reading experience while exploring different news categories.</p>
<br>
<h2>🚀 Key Features</h2>
<ul>
<li>Hero / Breaking News section highlighting trending or featured articles</li>
<li>News categories such as Politics, Technology, Entertainment, and Sports</li>
<li>Latest article section with image thumbnails, titles, and publication dates</li>
<li>Dedicated news detail pages for full article reading</li>
<li>Responsive navigation menu for smooth category browsing</li>
<li>Mobile-friendly layout optimized for comfortable reading</li>
</ul>
<br>
<h2>⚙️ Technologies Used</h2>
<p>This project is built using a frontend-focused architecture with an emphasis on readability and responsive performance :</p>
<ul>
<li>HTML5 : Article structure and multimedia content organization.</li>
<li>CSS3 : Layout design, typography, and responsive styling.</li>
<li>JavaScript : Interactive navigation and content behavior.</li>
<li>Deployment : Vercel for fast hosting and Git integration.</li>
</ul>
